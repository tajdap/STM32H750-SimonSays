# Simon Says - STM32H750B-DK
Seminarska naloga pri predmetu Organizacija računalnikov.

## Opis
Igra Simon Says implementirana na razvojni ploščici STM32H7. 
Igra uporablja LCD zaslon za prikaz in Touchscreen za interakcijo.

## Zagon
Projekt uvozite v STM32CubeIDE in naložite na ploščico STM32H750B-DK.
